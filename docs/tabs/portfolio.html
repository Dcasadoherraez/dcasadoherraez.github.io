Concurrent EKF SLAM

Implementation of an Extended Kalman Filter SLAM approach in 2D parallelizing the system and observing the changes in performance.

HILTI SLAM Challenge 2022

Our solution for the HILTI SLAM Challenge 2022 combines the results from a high-frequency camera tracking ORB-SLAM2 and a low-frequency LIDAR method A-LOAM.

Edge Image Segmentation in the Cloud

Implementation of a real-time image segmentation pipeline using the NVIDIA Jetson Nano as the camera capturing system and a cloud GPU server for inference.

Mixed Reality Therapy

5% of the population has fear of heights. In this project we explored as a team the development of a VR therapeutic application for acrophobia treatment with sensation augmentation.
